# Vocab2000FA (Android)

اپلیکیشن آموزش ۲۰۰۰ واژه پرتکرار انگلیسی (NGSL) برای مخاطب فارسی‌زبان.

## ویژگی‌ها
- ۴ سطح (هر سطح ۵۰۰ واژه)
- هر درس ۱۰ واژه (۲۰۰ درس)
- تعریف انگلیسی (easy English) + IPA تقریبی
- امکان علامت‌گذاری «یاد گرفتم» و «علاقه‌مندی»
- جستجو، مرور درس‌ها، و کوییز (بر اساس تعریف انگلیسی)
- TTS (تلفظ با Text-to-Speech اندروید)
- امکان اضافه‌کردن معنی فارسی توسط کاربر (در صفحه جزئیات واژه)

## دیتاست
- لیست و رتبه‌بندی: NGSL 1.2 (CSV)
- تعریف‌ها: NGSL 1.2 English definitions (XLSX)
- فونتیک IPA: تبدیل تقریبی از CMUdict (ARPABET -> IPA)

## اجرا
1) پروژه را در Android Studio باز کنید.
2) Gradle Sync
3) Run روی یک دستگاه/امولاتور

> نکته: معنی فارسی به صورت پیش‌فرض خالی است (قابل وارد کردن داخل خود اپ).
